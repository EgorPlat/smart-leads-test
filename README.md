# Smart Leads — Конструктор сценариев (тестовое)

Мини-прототип модуля «Конструктор сценариев» (Flow-схема) на React + TypeScript + Vite.

## Запуск

1. Установить зависимости:
   ```bash
   npm install
   ```
2. Dev-сервер:
   ```bash
   npm run dev
   ```
3. Сборка:
   ```bash
   npm run build
   npm run preview
   ```

## Стек и решения

- **React 18**, **TypeScript (strict)**, **Vite**.
- **react-router-dom** для маршрутов (`/` список, `/editor/:id` редактор).
- **reactflow** для редактирования графа: добавление/перемещение узлов, соединения, удаление.
- Хранение в **localStorage** с имитацией API (задержка 300–500мс).  
  Модуль: `src/services/scenarios.ts`.
- Типы: `src/types.ts` (`Scenario`, `ScenarioMeta` и т.д.).
- Простые компоненты и стили без лишних абстракций.

## Функциональность

- Список сценариев: вывод, создание, удаление, переход к редактированию.
- Редактор: добавление узлов «Действие»/«Условие», соединение, перетаскивание, удаление выбранного, сохранение.
- Обработка состояний загрузки и ошибок при списке и сохранении.

## Использование ИИ

Части, делегированные ИИ:
- Первичная инициализация проекта и файловая структура.
- Базовая реализация страниц и сервиса хранения.
- Черновая стилизация и README.

Проверка кода:
- Локальный просмотр UI, ручная проверка сценариев: создание, редактирование, соединения, удаление, сохранение.
- Проверка типов TypeScript и сборки Vite.

## Заметки

- При удалении узла автоматически удаляются его связи.
- Данные хранятся в браузере, очистка — через DevTools (Local Storage) или кнопкой «Удалить» в списке.
